# Class vs Object Patterns

| Snippet | Pattern | Type | Reason |
|---------|---------|------|--------|
| 01 | Singleton | Object | Instance-level state |
| 02 | Mediator | Object | Runtime object coordination |
| 03 | Facade | Object | Wraps subsystem objects |
| 04 | Factory | Class | Uses inheritance |
| 05 | Decorator | Object | Dynamic wrapping |
| 06 | Prototype | Object | Object cloning |
| 07 | Chain of Resp | Object | Runtime chain |
| 08 | Command | Object | Command objects |
| 09 | Flyweight | Object | Object sharing |
| 10 | Mediator | Object | Object communication |

> **Class Patterns:** Compile-time relationships via inheratance
> **Object Patterns:** Runtime relationships via composition
